# 🚀 My Portfolio Repository

Selamat datang di repository saya! 🎉 Repository ini berisi proyek portofolio lengkap, mencakup pengembangan web menggunakan HTML, CSS, JavaScript, PHP, dan implementasi REST API. Selain itu, terdapat beberapa praktikum sebagai dokumentasi pembelajaran saya.

---

## 📂 Struktur Folder
```plaintext
📁 portofolio/
    ├── index.html           # Beranda portofolio
    ├── about.html           # Halaman tentang saya
    ├── contact.html         # Halaman kontak
    ├── styles/
    │   ├── style.css        # File CSS utama
    └── scripts/
        ├── main.js          # File JavaScript utama

📁 rest-api/
    ├── api/
    │   ├── connection.php   # Koneksi database
    │   ├── read.php         # API untuk membaca data
    │   ├── create.php       # API untuk menambah data
    │   ├── update.php       # API untuk mengupdate data
    │   └── delete.php       # API untuk menghapus data

📁 praktikum/
    ├── praktikum-1-html/
    │   ├── tugas1.html      # Tugas HTML pertama
    │   └── tugas2.html      # Tugas HTML kedua
    ├── praktikum-2-css/
    │   ├── tugas1.css       # Tugas CSS pertama
    │   └── tugas2.css       # Tugas CSS kedua
    └── praktikum-3-js/
        ├── tugas1.js        # Tugas JavaScript pertama
        └── tugas2.js        # Tugas JavaScript kedua

📖 Fitur Utama

1. Portofolio Web

- Tampilan responsif menggunakan HTML, CSS, dan JavaScript.
- Halaman-halaman utama: beranda, tentang, dan kontak.

2. REST API dengan PHP

- CRUD (Create, Read, Update, Delete) menggunakan MySQL sebagai database.
- Endpoint API dibuat dengan standar REST.

3. Praktikum Pembelajaran

- Praktikum dasar HTML (struktur dokumen, form, tabel).
- Praktikum CSS (layout, styling, animasi sederhana).
- Praktikum JavaScript (DOM manipulation, event handling).

🛠️ Teknologi yang Digunakan

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP, REST API
- Database: MySQL
- Tools: Visual Studio Code, Postman, XAMPP/WAMP

📦 Cara Menjalankan Proyek

- Clone repository ini: git clone https://github.com/username/repository-name.git
- Jalankan server lokal (contoh: XAMPP atau WAMP).
- Import database dari file database.sql di folder rest-api/db/.
- Akses proyek di browser melalui URL: http://localhost/portfolio/index.html
